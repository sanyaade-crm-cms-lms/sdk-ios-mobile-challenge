Parameters

Path to SSL certificate. (e.g. [[NSBundle mainBundle] pathForResource:@"MyCertificateName" ofType:@"cer"]). This value must not be nil.

Discussion

Providing nil for a path is strictly prohibited. Also please note that SSL pinning is incompatible with synchronous networking requests, so client code MUST use the asynchronous APIs.
