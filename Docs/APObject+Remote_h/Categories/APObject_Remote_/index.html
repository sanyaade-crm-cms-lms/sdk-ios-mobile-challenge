APObject(Remote)

Extends Class:
APObject
Declared In:
APObject+Remote

Introduction

Adds remote CRUD capabilities to APObject instances.

Discussion

All public properties are passed on to the remote server. Make sure to set base URL for the remote API with [ APObject setBaseURL: ] prior to use. Use APObject(RemoteConfig) to set the primary key property.
