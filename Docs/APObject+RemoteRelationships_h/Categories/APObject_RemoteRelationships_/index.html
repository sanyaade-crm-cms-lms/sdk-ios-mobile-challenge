APObject(RemoteRelationships)

Extends Class:
APObject
Declared In:
APObject+RemoteRelationships

Introduction

Adds support for loading relationship properties.

Discussion

Relationship properties point to other APObject instances (has many, has one, belongs to).

valueForKeyPath:error:async:

Gets value for given key path which may include relationships. If a relationship needs to be loaded, final value is returned asynchronously. 

- (id)valueForKeyPath:(NSString *)keyPath error:(NSError **)error 
        async:(APObjectCallback)callback;
Parameters
keyPath

Key path to load.

error

Error that has occured while executing the synchronous portion of request, if any.

callback

Callback to be executed when operation completes, with final value and/or error.

Return Value

Currently available value at given key path.
