Introduction

Provides the means to register devices/channels and ability to send messages through APNS.

registerDevice:callback:

Registers a device to push notifications 

+ (void)registerDevice:(NSString *)deviceId callback:(APPushCallback)callback;
Parameters
deviceId

Device identifier. Nil values and empty strings will cause a failed response upon request completion.

callback

Callback to be execute after registration request is complete.

sendMessage:channelName:data:callback:

Sends a message to a desired channel. 

+ (void)sendMessage:(NSString *)payload channelName:(NSString *)channelName 
        iOSData:(APPushNotificationIOSData *)iOSData androidData:(APPushNotificationAndroidData *)androidData 
        callback:(APPushCallback)callback;
Parameters
payload

Message to be sent.

channelName

Name of subscribed channel to which message is to be sent. Nil values and empty strings will cause a failed response upon request completion.

iOSData

Object containing alert, sound, badge and expiry information for use with APNS messages.

androidData

Object containing collapse key, delay and TTL information for use with GSM messages.

callback

Callback to be execute after message request is complete.
